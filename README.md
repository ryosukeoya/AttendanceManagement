## About

-   勤怠管理アプリです。カレンダーから勤務時間の確認もできます⛱

## Usage

```
 composer install && npm install && cp .env.example .env && php artisan key:generate
```

```
 npm run dev
```

```
 php artisan serve
```

| 項目           | 技術         |
